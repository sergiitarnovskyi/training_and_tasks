Для запуска скрипта необдимы права root и установленный python3


1. Установка Python 3.6:

Для начала установим Python 3.6. В зависимости от вашей версии RHEL, вам может понадобиться включить определенные репозитории:

```bash
sudo yum install https://centos7.iuscommunity.org/ius-release.rpm
sudo yum update
sudo yum install python36u python36u-libs python36u-devel python36u-pip
```

Проверьте правильность установки Python, запустив:

```bash
python3.6 -V
```

2. Создание скрипта:

Теперь создадим Python скрипт. На вашей машине откройте текстовый редактор, например:

```bash
nano install_nginx.py
```

Скопируйте и вставьте код из репозитория. Сохраните и закройте файл.

4. Предоставление разрешений:

Для того чтобы файл мог выполняться как скрипт, необходимо предоставить соответствующие разрешения. Для этого можно использовать следующую команду:

```bash
chmod +x install_nginx.py
```

5. Запуск скрипта:

Теперь этот скрипт можно запустить с помощью Python 3.6, который мы установили ранее:

```bash
sudo python3 install_nginx.py
```

6. На вопросы скрипта отвечайте в соответствии с инструкциями. 

Хотя было бы лучше подошел репозиторий с Nginx в вашем распоряжении, этот скрипт должен успеть обработать множество пользовательских случаев. Если иметь в виду базовый вариант, Nginx доступен в основном репозитории EPEL. Если он не включен, вы можетe добавить его следующим образом:

```bash
sudo yum install epel-release
```

Теперь вы должны быть готовы к работе!
